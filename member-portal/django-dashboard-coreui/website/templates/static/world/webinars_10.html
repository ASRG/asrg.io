    When: June 18th, 2020 @ 16:00 GMT +0

    Where: LiveStream

    Who: Vangelis Stykas, Security Enthusiast and Researcher

    Presentation:

    "Attacking cars from the Cloud, the API way of Stealing Cars"

    This presentation will cover a variety of attacks that we did on various 3rd party car alarms , GPS trackers and immobilizers. We will cover techniques used and goals pursued after connecting. And how common API errors that happened in the 90s still live for 20 years and can be used to attack a smart car.

    About the Presenter:

    Vangelis Stykas
    Security Enthusiast and Researcher

    Vangelis is a specialist in API security, worked at some startups and managed API development for a large number of applications. Additionally he has spent several years in mobile & web app development, researched maritime satcomms security with several ship system vulnerabilities to his name and more recently moved into penetration testing. He has successfully hacked cars, ships, locks, smart homes amongst other IoT devices.
